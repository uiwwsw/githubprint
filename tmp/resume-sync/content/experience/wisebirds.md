# (주)와이즈버즈

## 신규 프로젝트 준비

- 기간: 2020.03 - 2020.05
- 역할: 프론트엔드 개발자
- 설명: EAS 신규 사업용 웹 플랫폼 기반 개발
- 기술 스택: Nuxt.js, TypeScript, SCSS

### 주요 업무

- 신규 사업용 웹서비스 초기 베이스 구조 개발
- 페이지, 레이아웃, 공통 컴포넌트 구현
- 로그인, 회원가입 등 기본 사용자 플로우 개발
- TypeScript 기반 프로젝트 구조 구성
- TypeScript 도입을 위한 내부 기술 공유 및 교육 진행

### 성과

- 신규 사업 검증을 위한 웹서비스 기반 마련
- TypeScript 도입 및 팀 개발 역량 강화

## 팀 프로젝트

- 기간: 2019.06 - 2020.11
- 역할: 개인 프로젝트 주도 개발
- 설명: 내부 개인 프로젝트 개발 (Wisebirds)
- 기술 스택: Vue.js, JavaScript, TypeScript, Chrome Extension

### 주요 업무

- 사내 북마크 관리용 Chrome Extension 개발
- Vue 기반 공통 UI 컴포넌트 라이브러리 개발
- Vue 프로젝트 보일러플레이트 제작
- 공통 개발 자산 정리 및 초기 세팅 표준화
- 반복 작업 축소 및 개발 생산성 개선

### 성과

- 사내 공통 도구 및 개발 자산 구축
- 개발 속도와 협업 효율 개선

- 기간: 2019.02 - 2019.04

- 기술 스택: Vue.js, SCSS

### 주요 업무

- 디자인 결과물 공유 웹서비스 클라이언트 페이지 개발
- 디자인 결과물 관리용 어드민 페이지 개발
- Vue 기반 공통 컴포넌트 구조 설계
- SCSS 기반 스타일 구조 정리

### 성과

- 클라이언트 및 어드민 페이지 구축과 서비스 런칭 수행

## 사내결재시스템의 FE 개발 및 런칭

- 기간: 2018.12 - 2019.02
- 역할: 프론트엔드 개발자
- 설명: 사내 결재 서비스 웹 애플리케이션 개발
- 기술 스택: React, Redux, SCSS

### 주요 업무

- 사내 결재 서비스 UI 및 클라이언트 로직 개발
- 결재 요청, 승인, 반려 등 결재 플로우 화면 구현
- 공통 레이아웃 및 재사용 가능한 UI 컴포넌트 구성
- Redux 기반 전역 상태 관리 구조 설계
- SCSS 기반 스타일 구조 정리
- 업무용 서비스에 맞는 화면 흐름 및 UX 구성

### 성과

- 사내 결재 프로세스를 웹으로 구현
- 기능 확장과 유지보수를 고려한 프론트엔드 구조 마련
