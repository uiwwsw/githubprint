# 주식회사세미티에스

## 컨베이어 클라이언트 웹서비스

- 기간: 2023.11 - 2024.01
- 역할: 프론트엔드 리더
- 설명: 컨베이어 클라이언트 웹서비스 개발
- 기술 스택: React, SWR, Tailwind CSS, Monorepo

### 주요 업무

- 반도체 개발 장비용 클라이언트 웹서비스 설계 및 개발
- 프론트엔드 아키텍처 및 공통 개발 기준 수립
- Clean Architecture 기반 구조 적용
- React, SWR 기반 데이터 통신 및 상태 관리 구조 설계
- Monorepo 기반 프론트엔드/백엔드 개발 환경 구축
- 공통 타입, 유틸, API 인터페이스 정리
- 로컬 개발 환경 표준화 및 초기 세팅 자동화
- Tailwind CSS 기반 UI 구조 정리

### 성과

- 확장성과 안정성을 고려한 클라이언트 웹서비스 구축
- Monorepo 기반 협업 효율 및 코드 재사용성 개선

## SP4 클라이언트 웹서비스

- 기간: 2023.08 - 2023.10
- 역할: 프론트엔드 리더
- 설명: SP4 클라이언트 웹서비스 개발
- 기술 스택: Next.js, SWR, Tailwind CSS

### 주요 업무

- 반도체 개발 장비용 클라이언트 웹서비스 설계 및 개발
- 프론트엔드 아키텍처 및 개발 기준 수립
- Clean Architecture 적용
- 페이지 및 레이아웃 구조 설계
- SWR 기반 데이터 패칭 구조 설계
- Tailwind CSS 기반 공통 UI 패턴 정리
- 팀 리소스 관리 및 일정 조율
- 코드 품질 관리 및 협업 프로세스 정리

### 성과

- 안정성과 확장성을 고려한 클라이언트 구조 구축
- 리소스 관리 및 표준화로 프로젝트 진행 안정성 확보
